lpod.scriptutils¶

class lpod.scriptutils.StdoutWriter(buf='')¶

Bases: StringIO.StringIO

Some proxy to write output to stdout in scripts. Because The zipfile module raises “IOError: [Errno 29] Illegal seek” when writing to stdout directly.
